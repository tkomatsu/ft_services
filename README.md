# ft_services
## Description
Setting up a multi-service cluster.
### services
* nginx
* MySQL
* WordPress
* phpMyAdmin
* grafana
* influxdb

## Requirement
* minikube
* kubectl
* Docker

## Usage
```shell
git clone git@github.com:tkomatsu/ft_services.git
cd ft_services
./setup.sh
```
